# Java Compiler Project 🚀

A fully functional compiler developed in **Java** as part of the Computer Engineering degree at **UIB**. This project covers the entire translation lifecycle, from source code analysis to executable code generation.

## 🛠️ Features
* **Front-end Analysis:** Implementation of lexical and syntactic analysis using **AST** (Abstract Syntax Tree).
* **Semantic Analysis:** Scope management and type checking through **Symbol Tables**.
* **Back-end:** Intermediate code generation and control flow optimization.
* **Target:** Final translation to **Assembly**, enabling the actual execution of the source code.
